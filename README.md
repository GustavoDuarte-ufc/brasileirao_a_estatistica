# Top clubes do Brasileirão Série A
Trabalho Final para a cadeira de Visualização de Dados. 
Uma página web com análise dos resultados do Campeonato Brasileiro de Futebol - Brasileirão Série A, entre os de 2021 à 2025.

Com o objetivo de responder às seguintes perguntas:
● Quais foram os principais clubes no Brasileirão, série A nestes últimos 5 anos?
● Existe uma correlação entre melhores jogadores do campeonato com o resultado do
seu clube na colocação?
● Clubes com maiores probabilidades de vencer o Campeonato Brasileiro de 2026?
● A chance de vitória no Brasileirão séria A, também tem relação com outros
campeonatos vencidos?
